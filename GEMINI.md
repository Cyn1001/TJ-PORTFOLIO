# Resident Evil Portfolio Project

## Architecture
- **Vanilla JS & CSS**: High performance, no heavy frameworks.
- **Three.js**: Used for the atmospheric background particle system with glitch effects.
- **Responsive Design**: Mobile-friendly grid for projects.

## Conventions
- **Aesthetic**: RE7/8 "Found Footage" style (Dark, VHS, Glitch).
- **Typewriter**: Use `.typewriter` class for auto-typing text.
- **Glitches**: Use `.glitch-text` with `data-text` attribute for CSS glitch effects.

## File Map
- `index.html`: Main content.
- `css/style.css`: All styling (Variables in `:root`).
- `js/app.js`: Three.js scene and UI interactivity.
- `images/`: Asset directory.
